# Website using react & laravel

- Setup Backend 
Directory structure 
```
backend
--laravel-backend

frontend
-- react-frontend
```

- Install Laravel
```
composer create-project laravel/laravel laravel-backend
```

- Install React via Vite
```
npm create vite@latest

  cd front-end
  npm install
  npm run dev
```
- Install bootstrap for  frontend
```
npm install react-bootstrap bootstrap
```
